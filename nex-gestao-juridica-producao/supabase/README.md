# Supabase - Nex Gestão Jurídica

1. Crie um projeto no Supabase.
2. Acesse SQL Editor.
3. Cole e execute `schema.sql`.
4. Copie a `Project URL`, `anon key` e a connection string PostgreSQL.
5. Preencha `.env.local` localmente e as variáveis na Vercel.

O app atual roda em modo demonstração/localStorage para facilitar apresentação, VS Code e Vercel. O schema já deixa a base pronta para persistência real.
