# Guia de Onboarding do Cliente v4.5

## 1. Pré-implantação

- Confirmar nome do escritório.
- Confirmar CNPJ/CPF.
- Confirmar e-mail do admin.
- Confirmar plano contratado.
- Explicar que a primeira fase é implantação assistida.

## 2. Cadastro da empresa

No Admin Master:

1. Acesse Empresas.
2. Clique em Nova Empresa.
3. Preencha dados do escritório.
4. Cadastre o admin responsável.
5. Copie a matrícula gerada.

## 3. Primeiro acesso

Enviar ao admin:

- link do sistema;
- e-mail de acesso;
- senha temporária/convite;
- matrícula da empresa;
- instruções de troca de senha.

## 4. Primeiros cadastros

Com o cliente em call ou presencial:

1. Cadastrar 3 clientes reais.
2. Cadastrar 3 processos reais.
3. Criar 3 tarefas.
4. Inserir 1 lançamento financeiro.
5. Liberar 1 documento no portal.
6. Gerar 1 relatório.

## 5. Validação

- Testar login do admin.
- Testar login de funcionário.
- Testar portal do cliente.
- Verificar se dados permanecem após sair e entrar.
