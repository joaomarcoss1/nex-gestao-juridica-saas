# Guia de Testes com Cliente Real v4.5

## Escopo recomendado

Comece com 1 a 3 escritórios em piloto assistido.

## Testes obrigatórios com o cliente

- Login do admin da empresa.
- Cadastro de cliente.
- Cadastro de processo.
- Criação de tarefa.
- Upload de documento.
- Lançamento financeiro.
- Relatório PDF.
- Acesso do cliente por nome/CPF.
- Uso no celular.

## Critérios para aprovar piloto

- O cliente consegue usar sem orientação a tela principal.
- Nenhum dado de outra empresa aparece.
- Dados salvos continuam após logout/login.
- Erros aparecem de forma amigável.
- Mobile funciona sem cortes.

## Coleta de feedback

Pergunte:

1. O que ficou confuso?
2. O que faltou para o dia a dia?
3. Qual tela pareceu mais útil?
4. O sistema aparenta valor suficiente para pagar mensalidade?
5. Qual preço faria sentido?
