# Dependências externas — v5.2

- Supabase Auth, Database, RLS, Storage e RPCs.
- Vercel para frontend e funções serverless.
- Stripe para assinatura e cobrança do SaaS.
- Google Calendar para sincronização externa de agenda.
- Google Leads/Ads para entrada real de leads.
- Serviço de e-mail do Supabase para link mágico do portal.
- APIs opcionais de tribunais, ICP, WhatsApp, Asaas e Mercado Pago.

As integrações permanecem desabilitadas ou em estado de prontidão quando suas credenciais não estão configuradas. O sistema não deve simular sucesso de serviço externo.
